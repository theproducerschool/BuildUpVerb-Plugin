# BuildUp Reverb

A unique reverb plugin that builds intensity over time, perfect for creating tension and dynamic effects in your productions.

## Features

- **Build Up Time**: Control how quickly the reverb intensity increases (0.1 - 10.0 seconds)
- **Target Level**: Set the maximum reverb level to build up to
- **Room Size**: Adjust the reverb space size
- **Damping**: Control high-frequency absorption
- **Width**: Adjust stereo spread
- **Dry/Wet Mix**: Blend between original and processed signal
- **Reset**: Instantly reset the build-up effect

## Building

This plugin requires JUCE framework. To build:

1. Clone this repository
2. Open in your preferred IDE with CMake support
3. Build the project

The plugin will be built as both AU and VST3 formats on macOS.
